Տեղադրել GitHub-ում

Ինչո՞ւ ընտրել մեր App-ը

✨

Ոճի Ավտոմատացում

Հեշտացրեք ձեր ստուդիայի աշխատանքային հոսքերը GitHub Action-ների միջոցով։

📅

Խելացի Ամրագրումներ

Ինտեգրեք ամրագրումների համակարգը անմիջապես ձեր GitHub Issue-ների հետ։

💬

Հաճախորդների Կապ

Ավելացրեք հաճախորդների հետադարձ կապի ավտոմատացված հաղորդագրություններ։

🌸

Գեղեցիկ Դիզայն

Լիովին համատեղելի է գեղեցկության ոլորտի բոլոր ստանդարտների հետ։

Տեղադրման Քայլերը

1

Այցելեք GitHub Marketplace

Սեղմեք «Տեղադրել GitHub-ում» կոճակին, որպեսզի գնաք App-ի էջ։

2

Ընտրեք Ձեր Կազմակերպությունը

Ընտրեք այն GitHub կազմակերպությունը կամ օգտատիրոջ հաշիվը, որտեղ ցանկանում եք տեղադրել App-ը։

3

Կարգավորեք Թույլտվությունները

Տվեք անհրաժեշտ թույլտվությունները App-ին, որպեսզի այն կարողանա աշխատել ձեր Issue-ների և PR-ների հետ։

4

Պատրաստ է

BLOOM MOOD App-ը այժմ ակտիվ է և պատրաստ է օգնել ձեր ստուդիայի կառավարմանը։
